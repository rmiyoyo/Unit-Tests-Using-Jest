# 📖 To-do List Project <a name="about-project"></a>

In this project, I will write a few practical tests for JavaScript functions using the Jest library. The AAA pattern is followed to make my tests easier for other developers to read and understand. The TDD approach in used.

. 

## 🛠 Built With <a name="built-with"></a>

<!-- Features -->

### Key Features <a name="key-features"></a>

- **A function stringLength(string) that takes any string as an argument and returns its characters count. The function can check if the string is at least 1 character long and not longer than 10 characters. Throw errors if those conditions are not met. A test for this function is implemented.**
- **A function reverseString(string) function. It should take a string as an argument and return it reversed. A test for this function is implemented.**
- **A a simple calculator class or object, which will have 4 methods: add, subtract, divide, and multiply.**
- **A test for a capitalize(string) function. The test should make sure that this function takes a string as an argument and returns that string with the first character capitalized.**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- A Chrome browser
- Node
- npm

### Setup

Clone this repository to your desired folder:


```sh
cd my-folder
git clone git@github.com:rmiyoyo/Unit-Tests-Using-Jest.git
```

### Install

Install the prerequisites this project using:

```sh
cd Unit-Tests-Using-Jest
npm init -y
npm install --save-dev hint@7.x
npm install --save-dev stylelint@13.x stylelint-scss@3.xstylelint-config-standard@21.x stylelint-csstree-validator@1.x
npm install --save-dev jest
```

### Usage

To run the project:

```bash
node index.js
```

### Run tests

To run tests after setting up the linters, run the following command:

```sh
npx hint .
```
```sh
npx stylelint "**/*.{css,scss}"
```
```bash
npx eslint .
```
```bash
node index.test.js
```
